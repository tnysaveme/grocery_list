# tny's grocery list

A simple, minimalist grocery list application connected to Spotify. It displays your recently played tracks and allows you to view them in a chronological list or categorized by album/playlist.

## Features

- **Spotify Integration**: Connects to your Spotify account to fetch recently played tracks.
- **Minimalist Design**: clean, black-and-white aesthetic.
- **Two Views**: Chronological list and categorized view.

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Set up environment variables in `.env.local` (Client ID, Client Secret, Redirect URI).
4.  Run the development server: `npm run dev`

## Tech Stack

- Next.js
- Tailwind CSS
- Framer Motion
- Spotify Web API
